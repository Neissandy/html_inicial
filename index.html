Hello World!!

Nesta atividade vamos aprender a usar o Git junto ao GitHub!

Pessoa 1:
Configure o git com seu nome de usuário e email;
Inicie um repositório local com nome html_inicial;
Crie uma página html com head, body e pelo menos um elemento h1 com o título "Hello World";
Adicione e comite o conteúdo do repositório;
Crie um repositório remoto no github com o mesmo nome do repositório local;
Adicione um remoto para o repositório github;
Pushe o conteúdo do repositório;
Adicione a Pessoa 2 como colaboradora no repositório remoto;
